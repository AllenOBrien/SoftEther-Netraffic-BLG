# SoftEther-Netraffic-BLG
SoftEther WEB端BLG流控

###使用方法
> 先安装lnmp,接着把流控放对应目录，打开流控监控。
> 一键脚本: cd / && wget https://raw.githubusercontent.com/wx1183618058/SoftEther-Netraffic-BLG/master/web.sh && chmod 777 web.sh && ./web.sh
