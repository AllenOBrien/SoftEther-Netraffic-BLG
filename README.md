# SoftEther-Netraffic-BLG
SoftEther WEB端BLG流控

###使用方法
> 先安装lnmp,接着把流控放对应目录，打开流控监控。 

###一键脚本
注：不一定是最新版，如果需要可自行同步源码

``` sh
cd / && wget https://raw.githubusercontent.com/wx1183618058/SoftEther-Netraffic-BLG/master/web.sh && chmod 777 web.sh && ./web.sh
```
